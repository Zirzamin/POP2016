\subsection{2i.2}

Først defineres \textit{\textbf{a}} som "Hello World"

Derefter printer jeg de første fire char i a, for til sidst at printe fra den sjette char til slut. 

\lstset{}
\begin{lstlisting}
	> let a = "Hello World";;
	
	val a : string = "Hello World"
	
	> a.[..4];;
	val it : string = "Hello"
	> a.[6..];;
	val it : string = "World"
	
	> a.[6..]+", "+a.[..4]+"!";;
	val it : string = "World, Hello!"

\end{lstlisting}
